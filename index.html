Application 6 — Lockheed Flight Control Proof-of-Concept

Author: Abdulrahman Fathalla

Project Overview

This project is a real-time embedded systems proof-of-concept built on the ESP32 with FreeRTOS, themed as flight control firmware for a Lockheed Martin pilot training simulator. It demonstrates deterministic timing on safety-critical tasks (eject input, flight envelope monitoring) while servicing a soft real-time Wi-Fi dashboard for a simulated ground station.

System Highlights

  • 5 FreeRTOS tasks + 1 ISR, mixed Hard and Soft real-time
  • 4 sync primitives: binary semaphore, counting semaphore, queue, mutex
  • Hardware interrupt-driven eject handler with sub-tick response
  • Live Wi-Fi telemetry dashboard with remote LED control
  • Logic analyzer trace verifies 1000 ms heartbeat period
  • End-to-end envelope-breach response measured at ~16 ms (50 ms hard deadline)
